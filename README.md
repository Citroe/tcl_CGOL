Pretty basic Conway's Game of Life in TCL. 
25 by 25 grid

Blinker and traveller starting seeds are provided but you can put in your own starting seed and check it out although some seeds like Methuselahs may
not behave as expected due to the fact that the grid is 25 x 25, you can change the grid size but take care of the UI resolution as it may break if
not scaled properly.
